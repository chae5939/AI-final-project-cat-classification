# Transfer Learning 기반 고양이 품종 분류 모델

## 프로젝트 개요

ResNet18 기반 Transfer Learning을 활용하여
고양이 품종 분류 모델을 구현하였다.

## 데이터셋

Oxford-IIIT Pet Dataset

사용 품종
- Bengal
- Maine Coon
- Persian
- Ragdoll
- British Shorthair

총 500장 사용

## 모델

ResNet18
Transfer Learning

## 결과

Validation Accuracy : 91%

## 추가 실험

실제 반려묘 6마리를 활용하여
모델 성능을 검증하였다.
